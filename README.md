# WAIS V3.3.0 Stable

Manual Control Edition

## 核心原則
- 一般持股核心數字直接維護
- 交易／配息紀錄不再自動覆寫主資料
- 2421 一般持股與員工信託完全分開
- 員工信託分為自提、公提、合計
- 信託成本均價由本金餘額、現金與股數自動推算
- 單一 localStorage 資料檔，避免 IndexedDB 升級問題
- JSON 匯出／匯入
