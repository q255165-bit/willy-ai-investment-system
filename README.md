# Willy AI Investment System｜V2.0.1

免費、可長期使用的個人投資管理 PWA。

## 已包含
- 動態新增與修改投資標的
- 同一標的多個持有來源
- 買進、賣出、初始持股、股息再投入等交易紀錄
- 配息新增、修改、刪除
- 現金股息與再投入股息分開統計
- IndexedDB 本機資料庫
- JSON 完整備份與還原
- 手機安裝為 PWA
- GitHub Pages 可直接部署

## 發布
Repository → Settings → Pages → Deploy from a branch → main / root

## 重要
投資資料只儲存在目前瀏覽器，不會上傳到 GitHub。請固定匯出 JSON 備份。
